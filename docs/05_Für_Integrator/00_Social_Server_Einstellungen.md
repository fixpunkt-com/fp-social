# Festlegen der Social Server URL
Sie können die URL des verwendeten Social Servers in den Extension-Einstellungen unter ``apiUrl`` einstellen. Bitte beachten Sie **unbedingt** den abschließenden Slash am Ende der URL!

Der Standardwert lautet ``https://social.fixpunkt.com/ ``.<br>
Den Dev-Server finden Sie unter ``https://dev.social.fixpunkt.com/ ``.
