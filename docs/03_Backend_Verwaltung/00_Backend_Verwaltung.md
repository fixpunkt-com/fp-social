# Backenverwaltung

## Zugänge und Accounts anlegen
Damit Sie auf die sozialen Netzwerke zugreifen können müssen Sie definieren über welchen Zugang Sie auf welche Accounts zugreifen möchten.

Dies ist leider derzeit noch nicht über ein Backend-Modul möglich sondern muss über das Listen-Modul erfolgen.

[Hier][access_account] finden Sie die einzelnen Schritte.

## Accounts synchronisieren, Posts verwalten, durchsuchen und Synchronisation verwalten
Über das Backend-Modul können Sie verschiedene Aufgaben komfortabel erledigen:

* **[Accounts synchronisieren und Posts verwalten][posts]**<br>
  Sie können Accounts synchronisieren, Posts einlesen, diese ein- und ausblenden sowie das Vorschaubild eines jeden Posts wählen.
* **[Posts durchsuchen][search]**<br>
  Sie können bereits eingelesene Posts durchsuchen.
* **[Synchronisation bearbeiten][sync]**<br>
  Die Einstellunge zur automatischen Synchronisation können hier bearbeitet werden. Zudem kann eingestellt werden, was mit neu eingelesenen Posts passieren soll.
