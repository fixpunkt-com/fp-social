# Posts verwalten
## Liste aller Accounts
![alt text][list]
Hier finden Sie eine Liste aller angelegten Accounts.

Sie können diese Liste im oberen Bereich der Tabelle nach Typ filtern.

Folgende Informationen werden Ihnen hier zur Verfügung gestellt:
* **Automatische Synchronisation:**<br>
  Ganz links sehen Sie zwei grüne oder gelbe Pfeile, welche einen Kreis bilden. Diese zeigen an ob die automatische Synchronisation dieses Accounts stattfindet (grüner Kreis) oder nicht (gelber Kreis). Die automatische Synchronisation verwalten Sie unter [Synchronisation][sync].
* **Automatisches Freischalten:**<br>
  Wenn eingelesene Posts automatisch freigeschaltet werden erscheint neben dem Hiweis zur automatischen Synchronisation ein grünes Häckchen, ansonsten ein rotes X.
* **Netzwerk:**<br>
  Das Netzwerk aus dem die Daten gezogen werden sollen.
* **Modus:**<br>
  Der Account aus welchem und der Modus in dem die Daten ausgelesen werden.
* **Anzahl Posts:**<br>
  Die Anzahl der bisher importierten Posts.
* **Zuletzt syncrhonisiert am...:**<br>
  Hier wird Ihnen angezeigt, wann das letzt mal versucht wurde, den Account zu synchronisieren. Sollte dies **nicht** erfolgreich gewesen sein, wird zusätzlich der Zeitpunkt der letzten erfolgreichen Synchronisation ausgebene sowie eine Fehlermeldung angezeigt.

Folgende Aktionen sind möglich:
* **Synchronisieren:**<br>
  Synchronisiert den Account. Teilweise öffnet sich ein weiteres Fenster in dem Sie weitere Informationen angeben können, was Sie synchronisieren möchten.
* **Beiträge anzeigen:**<br>
  Klicken Sie hier, wenn Sie alle bereits eingelesenen Beiträge anzeigen möchten.

## Account Ansicht
In der Account Ansicht sehen Sie alle mit dem Account eingelesenen Posts.

Hier haben Sie folgende Möglichkeiten:
* **Vorschaubilder:**<br>
  Sollten mehrere Vorschaubilder verfügbar sein, können Sie mit einem Klick auf das jeweilige Bild und dem Button "Dieses Bild als Vorschaubild auswählen" das aktuelle Bild als Vorschaubild auswählen.
* **Posts anzeigen:**<br>
  Mit einem Klick auf "Post öffnen" können Sie sich den Post im betreffenden Netzwerk anzeigen lassen.
* **Ein- & Ausblenden:**<br>
  Mit einem Klick auf "Diesen Post ausblenden / einblenden" können Sie den Post im Frontend aus- bzw. einblenden.
* **Synchronisieren:**<br>
  Mit einem Klick auf "Post neu synchronisieren" können Sie den Post neu einlesen.

Sollten Sie einen Post im Netzwerk gelöscht haben so können Sie diese Löschung übernehmen indem Sie den Post neu synchronisieren.
