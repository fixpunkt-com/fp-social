# Frontend Plugin
Um die eingelesenen Posts auszugeben nutzen Sie bitte das Plugin ``[FpSocial] Ausgabe von Sozialen Medien``.

## Ausgabemodus
Das Plugin bietet bietet drei verschiedene Ausgabemodi die [hier][ausgabemodus] genauer beschrieben sind.

## Filtern der Posts
In jedem Modus kann die Ausgabe im Reiter ``Allgemeine Einstellunge`` gefiltert werden. Folgende Filter stehen zur Verfügung:
* **Ausgabe beschränken auf folgende Hashtags...**<br>
  ![alt_text][filter_hashtags]<br>
  Filtert die Posts nach Hashtags. Nur Posts, welche die ausgewählten Hashtags besitzen werden auch angezeigt. Dieser Filter wird nur dann berücksichtigt, wenn mindestens ein Hashtag ausgewählt ist.

## Darstellung der Posts
Unabhängig vom gewählten Modus kann die Darstellung der Posts im Reiter ``Darstellung Posts`` angepasst werden.<br>
Weitere Informationen zu den Anpassungsmöglichkeiten finden Sie [hier][darstellung_posts].
