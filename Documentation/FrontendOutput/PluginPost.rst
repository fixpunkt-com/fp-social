=====================
Plugin "Output Post"
=====================

The ``[FpSocial] Output Post`` plugin outputs a single, fixed selected post. It
is suitable for embedding a specific post on a page in a targeted way.

Settings
========

``Post``
    The post to be output. You can search by the ``Post Id``.

Display of the Posts
====================

How the selected post is displayed can be adjusted via the settings described
under :ref:`post-display`.
